# Очки и система зачёта

Очки начисляются по результатам гонки:

| Место | Очки |
|-------|------|
| 1     | 25   |
| 2     | 18   |
| 3     | 15   |
| 4     | 12   |
| 5     | 10   |
| …     | …    |
| 10    | 1    |

+1 очко за самый быстрый круг (если пилот в топ-10).

Существуют два зачёта:
- **Личный** — для пилотов
- **Командный** — суммарно для команды
